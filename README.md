# WIZ SALES – Premium Product Catalog

## Features
- Glassmorphism modern UI, fully responsive
- Firebase Auth, Firestore, Storage
- Admin CRUD with drag‑and‑drop image upload
- Live search, filters, sorting
- WhatsApp buy button, QR codes, product sharing
- PWA ready

## Setup
1. Replace `firebaseConfig` in `firebase-init.js`
2. Enable Email/Password auth in Firebase Console
3. Create an admin user manually
4. Deploy to Firebase Hosting or any static host
